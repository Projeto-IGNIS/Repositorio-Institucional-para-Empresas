# Documentação do Projeto

Esta pasta contém toda a documentação técnica e de usuário do sistema.

## Estrutura planejada

```text
docs/
├── architecture/
│   ├── system-design.md
│   ├── database-schema.md
│   └── api-documentation.md
├── user-guide/
│   ├── installation.md
│   ├── user-manual.md
│   └── admin-guide.md
├── development/
│   ├── setup-dev-environment.md
│   ├── coding-standards.md
│   └── testing-guide.md
└── deployment/
    ├── production-setup.md
    ├── monitoring.md
    └── troubleshooting.md
```

## Tipos de documentação

- **Arquitetura**: Documentação técnica do sistema
- **Guia do usuário**: Manuais para usuários finais
- **Desenvolvimento**: Guias para desenvolvedores
- **Deployment**: Documentação de produção e operações

## Formato

- Documentos em Markdown (.md)
- Diagramas usando Mermaid ou Draw.io
- Screenshots e imagens em pasta assets/
