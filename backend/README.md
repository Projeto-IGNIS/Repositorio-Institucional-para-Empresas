# Backend - Spring Boot

Esta pasta contém o código do backend da aplicação, desenvolvido com Spring Boot.

## Estrutura planejada:

```
backend/
├── src/
│   └── main/
│       ├── java/
│       │   └── com/institucional/
│       │       ├── controller/
│       │       ├── service/
│       │       ├── repository/
│       │       ├── model/
│       │       ├── dto/
│       │       ├── config/
│       │       └── security/
│       └── resources/
│           ├── application.yml
│           └── application-prod.yml
├── pom.xml
└── Dockerfile
```

## Tecnologias:

- Spring Boot
- Spring Security
- Spring Data JPA
- Spring Data MongoDB
- JWT
- PostgreSQL
- MongoDB
- Redis
